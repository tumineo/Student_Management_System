#  Student Market

A desktop-based e-commerce application that has user management feature developed in C# and SQL Server that enables students to buy and sell products within a secure marketplace. 
The system allows users to create accounts, list products for sale, browse available items and purchase products from other students.

---

##  Overview

Student Market was developed to provide a simple and secure platform where students can trade products within their varsity. 
The application includes user authentication, product management, user management and shopping features 
demonstrating core software development principles such as database integration, CRUD operations and object-oriented programming.

---

##  Features

###  User Management(admin)
- Create a new account
- Secure user login
- User authentication
- Manage personal profile

###  Marketplace
- Browse products
- Search for products
- View product details
- Buy products
- Sell products

###  Product Management
- Add new products
- Edit product listings
- Delete products
- Manage available inventory

###  Database
- Store user information
- Store product information
- Manage transactions using SQL Server
